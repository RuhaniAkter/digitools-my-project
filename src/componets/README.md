# 🛠️ Project Name: My Project Digitools

## 📄 Description

A modern React web application to browse and manage digital tools with a dynamic shopping cart experience.

## 🚀 Technologies Used

- React.js
- Tailwind CSS & DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON

## ✨ Features

- **Live Cart Count**
  Automatically updates when you click **"Buy Now"**.

- **Cart Management**
  Easily remove individual items or clear the entire cart during checkout.

- **Interactive UI**
  Real-time toast notifications and dynamic button status updates for a smooth user experience.
